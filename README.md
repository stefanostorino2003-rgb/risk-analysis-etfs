ETF Risk Analysis Project
This project extracts various risk indicators from time series of ETFs using python:
- Mean
- Standard deviation
- Semi deviation
- Sharpe ratio
- Sortino ratio
- Max drawdown
- Drawdown duration
- Skewness
- Kurtosis
- Historic VaR
- Gaussian VaR
- Cornish-Fisher Var
- Historic CVar
I normalized these metrics in excel and i estimated a risk score of these ETFs
