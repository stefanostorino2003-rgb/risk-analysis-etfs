# ETF Risk Analysis Project

This project extracts various risk indicators from time series of ETFs using Python.

## Metrics
- Mean  
- Standard deviation  
- Semi deviation  
- Sharpe ratio  
- Sortino ratio  
- Max drawdown  
- Drawdown duration  
- Skewness  
- Kurtosis  
- Historic VaR  
- Gaussian VaR  
- Cornish-Fisher VaR  
- Historic CVaR  

## Risk Score
I normalized these metrics in Excel and estimated a risk score for the ETFs.
